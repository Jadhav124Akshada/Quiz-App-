## ğŸ Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional software installation required

### Installation
1. **Fork this repository**
2. **Clone it to your local system:**
   ```bash
   git clone https://github.com/Jadhav124Akshada/Quiz-App-.git
   ```
3. **Navigate to the project directory:**
   ```bash
   cd Quiz-App-
   ```
4. **Open `index.html` in your web browser**

### Usage
1. **Choose a Quiz Category** - Select from 11 different programming topics
2. **Start the Quiz** - Click the "Start" button to begin
3. **Answer Questions** - Select your answers and click "Check Answer"
4. **View Results** - See your score and performance statistics
5. **Track Progress** - Visit your profile to see achievements and history
